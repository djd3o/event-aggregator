# event-aggregator

A simple web app that aggregates events from platforms like Bandsintown, Songkick, Eventbrite, Resident Advisor, Dice, and Posh.

Built as a personal project to practice:

- React
- TypeScript
- Node.js
- Express
- API integration
- Web scraping
- Full-stack architecture

---

## Features

- View events from multiple platforms in one place
- Responsive event cards
- Search and filter events
- Simple modern UI with Tailwind CSS and shadcn/ui
- Mock data support for development
- Expandable architecture for future APIs and scrapers

---

## Tech Stack

### Frontend

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

### Backend

- Node.js
- Express
